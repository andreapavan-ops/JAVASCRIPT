Questa versione esegue FizzBuzz da 1 a 30 automaticamente al caricamento della pagina.
L'output è visibile nella console del browser (F12).

Nella programmazione, i % 3 === 0 è un modo comune per verificare se un numero è divisibile per 3.

1. L'operatore Modulo (%)

Il simbolo % calcola il resto di una divisione.

  • 6 ÷ 3 = 2 con resto 0 → 6 % 3 === 0
  • 7 ÷ 3 = 2 con resto 1 → 7 % 3 === 1

2. Il confronto (=== 0)

Il triplo uguale è un controllo di "stretta uguaglianza". Chiedere se il resto è esattamente 0 equivale a chiedersi: "Il 3 è entrato perfettamente in questo numero?"

Casi d'uso comuni

  • FizzBuzz: stampa "Fizz" se un numero è divisibile per 3.
  • Layout a griglia: ogni terzo elemento inizia una nuova riga.
  • Modelli alternati: applica uno stile a ogni terzo elemento di un elenco.
